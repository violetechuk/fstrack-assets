# FS Track GPL v2 Assets
This repository hosts the assets we used from [https://github.com/ADSBexchange/tar1090](https://github.com/ADSBexchange/tar1090) under the GPL v2 license.

The following assets were **not modified** by us for use with FS Track, and were used directly from the repository above:

- **Aircraft silhouettes** (`images/aircraft/sil`)
- **Country flags** (`images/flags`) - **Note:** Some of the files in this directory have been renamed for better use with FS Track.
- **Altitude gradient SVG (in feet)** (`images/altitude.svg`)

The following assets were **modified** by us for use with FS Track:

- **Aircraft icons** (`images/aircraft`) - Originally the icons were hosted within the `markers.js` file. We extracted the SVG path data from this file for the icons we needed, and modified them to suit our use of them.

This repository will be kept up-to-date as much as possible to reflect FS Track's use of the assets.
